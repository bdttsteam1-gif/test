매달 Sales Report 반영 방법
  1. 아래에 이번 달 _Boditech__Sales_Report_20XX_M_XXX.xlsm 파일을 업로드합니다.
  2. 파일 안의 C01 시트를 자동으로 읽어, 파일에 포함된 연도·월 데이터만 교체(증분 업데이트)합니다. 기존 다른 연도·월 데이터는 그대로 유지됩니다.
  3. 업데이트된 sales-data.js 파일을 다운로드해 GitHub의 html 폴더에 덮어쓰면 모든 직원에게 반영됩니다.
※ 월 보고서는 연초부터 해당 월까지 누적(YTD) 형식이므로, 파일에 있는 월은 전부 최신 값으로 교체됩니다. 같은 파일을 여러 번 올려도 중복되지 않습니다.
